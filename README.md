🌤 Weather App

A modern Weather App built with HTML, CSS, and JavaScript using the OpenWeatherMap API.
Get real-time weather information for any city, view 5-day forecasts, and switch between light and dark modes.

🔹 Features

Search weather by city name

Use your current location to get weather automatically

Display current temperature, humidity, wind speed, and “feels like” temperature

5-day forecast with weather icons and day names

Dynamic backgrounds based on weather conditions

Light/Dark mode toggle with icon change

Recent searches saved in local storage

Loading spinner while fetching data

Glassmorphism UI design for modern look

Fully responsive on mobile devices

🔹 Technologies Used

HTML5

CSS3 (Glassmorphism + responsive design)

JavaScript (ES6)

OpenWeatherMap API
